# html-template

自分が学習用のWEBアプリを作る際の、ベースになるHTML

## ファイル説明

```cmd
html-template
├─ajaxResponse
│      base-table.json          ......base.htmlのAjaxによるテーブル表示向けデータ
│
├─static
│  ├─css
│  │      common.css            ......共通部品のcss
│  │
│  └─js
│          base.js              ......base.html用のJavaScript
│          myGridjsConfig.js   ......テーブル用ライブラリGrid.js向けの日本語表示用設定値
│
└─template
        base.html               ......「UI部品基本設計」としてのHTML
```
